# Blog generated from md files

[Example here](https://events.novavovikov.ru/)

## Installation
You will need Docker to run site lifecycle tasks

## Run

### dev mode
```shell script
./run-dev.sh
```

> `backend` - http://localhost:5000 
>
> `frontend` - http://localhost:3000


### prod mode
backend with proxy:
```shell script
./run-prod.sh
```
